# Interview Scheduler

## Setup

Install dependencies with `npm install`.

## Running Webpack Development Server

```sh
npm start
```

## Running Jest Test Framework

```sh
npm test
```

## Running Storybook Visual Testbed

```sh
npm run storybook
```


## Dependencies
  - axios
  - classnames
  - normalize.css
  - react
  - react-dom
  - react-scripts
  - ws
